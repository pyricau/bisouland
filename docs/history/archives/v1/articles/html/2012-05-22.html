Bonjour et bienvenue sur Bisouland v2 !

Bisouland est un jeu gratuit par navigateur.
Volez des points d'amour de vos adversaires en leur envoyant des bisous !

Ce site est en perpétuelle évolution et en cours de développement, n'hésitez pas à revenir régulièrement pour voir les nouveautés.

Bisouland version 2.1.0

Le 14/05/2012

Comme promis, nous ajoutons régulièrement des nouveauté à Bisouland !

Fonctionnalités ajoutées

Lors de la version 2.0.0, en tappant le Konami code on accédait à la page listant la population de Bisouland. Le code permettait également de générer un nouveau personnage qui venait grandir cette popuation.
Cette version étant moins technique, il n'y aura pas d'easter egg pour accéder aux nouvelles fonctionnalités que voici :

  • Population accessible via le menu ;
  • nouveau design, fonctionnel sur toutes plateformes ;
  • une visite sur le site fera augmenter la population ;
  • les personnages ont désormais des Points d'Amour ;
  • les PA diminuent automatiquement d'une unité par seconde ;
  • lorsqu'ils atteignent 0, le personnage perd le jeu et est automatiquement supprimé.

Fin de la v1

La v2 est désormais disponible depuis l'adresse normale, ce qui signifie que la v1 est terminée : merci à tous d'avoir joué, et félicitations aux vainqueurs !

Bisouland version 2.0.0

Le 02/05/2012

Voici enfin la version tant attendue : Bisouland v2 !
C'est une refont complète du jeu, donc attendez vous à du nouveau !
